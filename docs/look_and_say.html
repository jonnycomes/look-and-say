look_and_say

A python module for exploring look and say sequences in the spirit of John H Conway.

The following assumes familiarity with the terminology and notation for look and sequences introduced by Conway in his delightful article The Weird and Wonderful Chemistry of Audioactive Decay.

This module can be used to recover several of Conway's results on standard look and say sequences. Additionally, it can be used to discover new results on various nonstandard look and say sequences. For an introduction to nonstandard look and say sequences, see the notes here.

Example Sessions

Standard decimal

The following session illustrates how the look_and_say module can be used to recover some of Conway's results.

>>>

... decimal = LookAndSay()
>>> 
>>> # Perform the fundamental look and say operation:
... decimal.say_what_you_see('1222111')

>>>

... decimal.generate_sequence(seed='1', terms=5)

Gray code

The following session shows how to use the module to explore a nonstandard look and say sequence. We use the binary number system known as Gray code to generate the sequence. The corresponding LookAndSay object depends on the say function which converts a positive integer into its Gray code.

>>> from look_and_say import *
>>>
>>> # Define the "say function"
... def gray(num):
...     assert num < 8, "This say function can only count to 7."
...     gray_code = {1:'1', 2:'11', 3:'10', 4:'110', 5:'111', 6:'101', 7:'100'}
...     return gray_code[num]
... 
>>> # Create the LookAndSay object and generate a look and say sequence
... gray_ls = LookAndSay(gray)
>>> gray_ls.generate_sequence(seed='0', terms=6)
>>> gray_ls.get_sequence()
['0', '10', '1110', '10110', '111011110', '10110110110']
>>> 
>>> # Use a BinaryChemistry object to determine the chemical properties
... gray_chem = BinaryChemistry(gray_ls)
>>> gray_chem.generate_elements('0')
>>> gray_chem.print_periodic_table()
element   string   abundance    decay
E1        10       0.0          [E3]
E2        110      58.5786438   [E4]
E3        1110     0.0          [E1, E2]
E4        11110    41.4213562   [E2, E2]
>>> 
>>> # The maximal real eigenvalue of the decay matrix gives the long term
... # growth rate of look and say sequences.
... gray_chem.get_max_eigenvalue()
1.4142135623730958
>>> 
>>> # The growth rate is the maximal real root of the characteristic polynomial
... gray_chem.get_char_poly()
(lambda - 1)*(lambda + 1)*(lambda**2 - 2)

Standard ternary

The following session illustrates how to use the module to explore look and say sequences using the standard ternary number system (i.e. using base 3 with digits 0, 1, and 2). The results are similar to those discussed here.

To construct the corresponding LookAndSay object requires a say function which, in this case, is a function that converts an integer to it's standard ternary representation. The construction of the Chemistry object requires both the LookAndSay object as well as a splitting function which determines when the terms of the look and say sequences split. In this case, one can show that the terms will always split after a run of 0's as well as between a 2 (on the left) and either a 10 or a 1110 (on the right). We use a SplitFuncFactory object to create the splitting function. 

>>> from look_and_say import *
>>> 
>>> # Define a "say function"
... def ternary(num):
...     '''Returns the ternary representation of a nonnegative integer'''
...     if num < 3:
...         return str(num)
...     return ternary(num // 3) + str(num % 3)
...  
>>> # Use the Split Function Factory to create a split function:
... sff = SplitFuncFactory()
>>> sff.declare_split_after('0')
>>> sff.declare_splitting_pairs(('2', '1110'), ('2', '10'))
>>> split = sff.get_split()
>>> 
>>> # Instantiate the LookAndSay and Chemistry objects:
... ternary_ls = LookAndSay(ternary)
>>> ternary_chem = Chemistry(ternary_ls, split)
>>> 
>>> # Generate elements and order them according to relative abundances:
... ternary_chem.generate_elements('0', '1', '2')
>>> ternary_chem.order_elements('abundance')
>>>

  17    """
  18    Splits a string into a list of substrings according to Conway's Splitting Theorem.
  19    Assumes the string is not empty. 
  20
  21    ## Example Session:
  22    ```python

  30    """
  31    chunks = []
  32    start = 0
  33    for i in range(1, len(string)):
  34        if _is_split_pair_Conway(string[start:i], string[i:]):
  35            chunks.append(string[start:i])
  36            start = i
  37    chunks.append(string[start:])
  38    return chunks

  40def _is_split_pair_Conway(L, R):
  41    """Implementation of Conway's Splitting Theorem"""
  42    if L == '' or R == '':
  43            return True # pragma: no cover
  44    if L[-1] not in ['1', '2', '3'] and R[0] in ['1', '2', '3']: # n] and [m
  45            return True
  46    if L[-1] == '2': # 2]
  47            if len(R) > 1 and R[0] == '1' and R[1] != '1' and (len(R) == 2 or R[2] != R[1]): #[1^1X^1
  48                    return True
  49            if len(R) > 2 and R[0] == '1' and R[1] == '1' and R[2] == '1' and (len(R) == 3 or R[3] != '1'): #[1^3
  50                    return True
  51            if R[0] == '3' and (len(R) == 1 or (R[1] != '3' and (len(R) < 4 or R[2] != R[1] or R[3] != R[1]))): #[3^1X^\not=3
  52                    return True
  53            if R[0] not in ['1', '2', '3'] and (len(R) == 1 or R[1] != R[0]): #[n^1
  54                    return True
  55    if L[-1] != '2' and len(R) > 1 and R[0] == '2' and R[1] == '2': # \not=2]
  56            if len(R) > 3 and R[2] == '1' and R[3] != '1' and (len(R) == 4 or R[4] != R[3]): #[2^21^1X^1
  57                    return True
  58            if len(R) > 4 and R[2] == '1' and R[3] == '1' and R[4] == '1' and (len(R) == 5 or R[5] != '1'): #[2^21^3
  59                    return True
  60            if len(R) > 2 and R[2] == '3' and (len(R) == 3 or (R[3] != '3' and (len(R) < 6 or R[4] != R[3] or R[5] != R[3]))): #[2^23^1X^\not=3
  61                    return True 
  62            if len(R) == 2 or (len(R) > 2 and R[2] not in ['1', '2', '3'] and (len(R) == 3 or R[3] != R[2])): #[2^2n^(0 or 1)
  63                    return True
  64    return False

 200    """
 201    def __init__(self, say = None):
 202        super(LookAndSay, self).__init__()
 203        self._is_Conway = False
 204        if say == None:
 205            say = (lambda n : str(n))
 206            self._is_Conway = True
 207        self.say = say
 208        self.sequence = []
 209
 210    def _chunk_op(self, char_count, char):
 211        """
 212        Conversion of the say function to a two parameter function 
 213        if the say function takes only one parameter; 
 214        just a copy of the say function otherwise.
 215        """
 216        try:
 217            return self.say(char_count, char)
 218        except:
 219            return self.say(char_count) + char
 220
 221    def get_sequence(self):
 222        """Returns the look and say sequence as a list of strings"""
 223        return self.sequence
 224
 225    def say_what_you_see(self, string):
 226        """
 227        The fundamental look and say operation that generates each 
 228        term of a look and say sequence from its predecessor. For example, 
 229        using the standard (default) LookAndSay object, 
 230        ``say_what_you_see('1112222333')`` returns ``'314233'``.
 231        """
 232        if not string: return '' # handles empty string, which is falsy
 233        letter = string[0]
 234        result = ''
 235        count = 0
 236        for ch in string:
 237            if ch == letter: 
 238                count += 1
 239            else:
 240                result += self._chunk_op(count, letter)
 241                count = 1
 242                letter = ch 
 243        result += self._chunk_op(count, letter)
 244        return result
 245     
 246    def generate_sequence(self, seed, terms):
 247        """
 248        Generates the look and say sequence. The parameter ``seed`` is 
 249        the initial term in the sequence, and ``terms`` is the 
 250        number of terms generated.
 251        """
 252        if not seed: return None # handles empty seed, which is falsy
 253        result = [seed]
 254        for _ in range(terms-1):
 255            result.append(self.say_what_you_see(result[-1]))
 256        self.sequence = result
 257
 258    def get_length_ratios(self):
 259        """
 260        Returns a list of the ratios of lengths of 
 261        successive terms in the look and say sequence.
 262        """
 263        terms = len(self.sequence)
 264        assert terms > 1, 'Look and say sequence does not have enough terms to compute the ratio of lengths.'
 265        return [len(self.sequence[i+1]) / len(self.sequence[i]) for i in range(terms - 1)]
 266
 267    def get_last_length_ratio(self):
 268        """
 269        Returns the ratio of the lengths of the last 
 270        two terms of the look and say sequence
 271        """
 272        return self.get_length_ratios()[-1]

 623    def clear_elements(self):
 624        """Resets the list of elements back to the empty list."""

 627    def _split_to_elements(self, string): 
 628        return [Element(chunk, self.las) for chunk in self.split(string)]

 630    def _generate_all_elements(self, strings):
 631        for string in strings:
 632            for elt in self._split_to_elements(string):
 633                if elt not in self.elements:
 634                    # add elt to the chemistry:
 635                    self.elements.append(elt)
 636                    # recursively set the decay for elt:
 637                    decay_elts = self._split_to_elements(self.las.say_what_you_see(elt.get_string()))
 638                    self._generate_all_elements(map(lambda e : e.get_string(), decay_elts))
 639                    elt._set_decay(decay_elts)
 640        # clean up decay for all elements:
 641        for elt in self.elements:
 642            dec = []
 643            for d in elt.get_decay():
 644                dec += [e for e in self.elements if e == d]
 645            elt._set_decay(dec)

 647    def _remove_intermittent_elements(self): 
 648        while True:
 649            common_elements = []
 650            for elt in self.elements:
 651                for d in elt.get_decay():
 652                    if d not in common_elements:
 653                        common_elements.append(d)
 654            if len(common_elements) < len(self.elements):
 655                self.elements = list(common_elements)

 685    def get_decay_matrix(self):
 686        """
 687        Returns the decay matrix as a nested list of integers 
 688        (i.e. a list of the rows).
 689        The order of the columns and rows correspond to the order
 690        in the list of elements.
 691        """
 692        mat = []
 693        e = self.elements
 694        for i in range(len(e)):
 695            row = []
 696            for j in range(len(e)):
 697                row.append(e[j].get_decay().count(e[i]))
 698            mat.append(row)
 699        return mat

 701    def get_max_eigenvalue(self):
 702        """
 703        Returns the maximal real eigenvalue of the decay matrix.
 704        In the standard case, this will give Conway's constant. 
 705        In general, this will give the growth rate of the look and say sequence. 
 706        This method assumes the existence of a real eigenvalue which
 707        is larger than (the absolute value) of every other eigenvalue.
 708        This assumption is usually guaranteed by the Perron-Frobenius Theorem.
 709        """
 710        assert len(self.elements) > 0, "The get_max_eigenvalue method requires a nonempty list of elements.\n\tTo fix: Use the generate_elements method prior to calling get_max_eigenvalue."
 711        eigenstuff = numpy.linalg.eig(numpy.array(self.get_decay_matrix()))
 712        eigenvalues = eigenstuff[0]
 713        return max(eigenvalues).real

 715    def get_char_poly(self, factor = True, latex = False):
 716        """
 717        Returns the characteristic polynomial of the decay matrix using sympy.
 718        By default the returned polynomial will be factored. 
 719        Use ``factor = False`` to get the expanded (i.e. unfactored) polynomial. 
 720        Use ``latex = True`` to return the polynomial formatted in latex.
 721        """
 722        chi = sympy.Matrix(self.get_decay_matrix()).charpoly()
 723        if factor:
 724            chi = sympy.factor(chi.as_expr())
 725        else:
 726            chi = chi.as_expr()
 727        if latex:
 728            return sympy.latex(chi)
 729        else:
 730            return chi

 732    def _get_abundances(self, dec_places = 7, abundance_sum = 100):
 733        """
 734        Returns a list of relative abundances of each element.
 735        By default the abundances are given as percentages, 
 736        so they will differ from Conway's abundances by a factor of \\(10^4\\).
 737        The abundances can be renormalized by setting the parameter ``abundance_sum``.
 738        The order of the list corresponds to the order of the list of elements.
 739        """
 740        eigenstuff = numpy.linalg.eig(numpy.array(self.get_decay_matrix()))
 741        eigenvalues = eigenstuff[0]
 742        eigenvectors = eigenstuff[1]
 743        index = numpy.where(eigenvalues == max(eigenvalues))
 744        limiting_eigenvector_nparray = eigenvectors[:,index].real
 745        # The next two lines are converting the numpy array to a list
 746        limiting_eigenvector = limiting_eigenvector_nparray.tolist()
 747        limiting_eigenvector = [elt[0][0] for elt in limiting_eigenvector]
 748        abundance = [abs(round(abundance_sum * num / sum(limiting_eigenvector), dec_places)) for num in limiting_eigenvector]
 749        return abundance

 786        """
 787        Reorders the list of elements depending on the parameter ``order_on`` as follows:
 788
 789        * ``order_on='abundance'``: Orders elements from highest abundance to lowest.
 790        * ``order_on='string'``: Orders elements according to the lexicographic order of their strings.
 791        * ``order_on='string length'``: Orders elements according to the lengths of their strings from shortest to longest.
 792        * ``order_on='name'``: Orders elements alphabetically according to their names.
 793        * ``order_on='key'``: Orders elements according to the function specified by the parameter ``key``.
 794
 795        You can reverse the ordering above by passing the extra parameter ``reverse = True``.
 796        
 797        By default this method will automatically rename the elements according to their new order.
 798        This will not happen if the elements are named via Conway or if the parameter ``rename = False`` is passed.
 799        """
 800        assert order_on in ['abundance', 'name', 'string', 'string length', 'key'], "Invalid parameter passed to order_elements. Valid parameter are 'abundance', 'name', 'string', 'string length', and 'key'."
 801        pt = self.get_periodic_table()

 803            'abundance': lambda e : pt[e.get_name()]['abundance'],
 804            'name': lambda e : e.get_name(),
 805            'string': lambda e : e.get_string(),
 806            'string length': lambda e : len(e.get_string()),
 807            'key': key
 808        }
 809        self.elements = sorted(self.get_elements(), key = sorted_key[order_on])
 810        if order_on == 'abundance':
 811            self.elements.reverse()
 812        if reverse:
 813            self.elements.reverse()
 814        if not self.las._is_Conway and rename:
 815            self._name_elements()

612    def __init__(self, las, split = split_Conway, elements = None):
613        super(Chemistry, self).__init__()

615        self.split = split

620    def get_elements(self):
621        """Returns the elements as a list."""
622        return self.elements

624    def clear_elements(self):
625        """Resets the list of elements back to the empty list."""
626        self.elements = []

628    def _split_to_elements(self, string): 
629        return [Element(chunk, self.las) for chunk in self.split(string)]

631    def _generate_all_elements(self, strings):
632        for string in strings:
633            for elt in self._split_to_elements(string):
634                if elt not in self.elements:
635                    # add elt to the chemistry:
636                    self.elements.append(elt)
637                    # recursively set the decay for elt:
638                    decay_elts = self._split_to_elements(self.las.say_what_you_see(elt.get_string()))
639                    self._generate_all_elements(map(lambda e : e.get_string(), decay_elts))
640                    elt._set_decay(decay_elts)
641        # clean up decay for all elements:
642        for elt in self.elements:
643            dec = []
644            for d in elt.get_decay():
645                dec += [e for e in self.elements if e == d]
646            elt._set_decay(dec)

648    def _remove_intermittent_elements(self): 
649        while True:
650            common_elements = []
651            for elt in self.elements:
652                for d in elt.get_decay():
653                    if d not in common_elements:
654                        common_elements.append(d)
655            if len(common_elements) < len(self.elements):
656                self.elements = list(common_elements)

660    def generate_elements(self, *seeds, reset = True):
661        """
662        Collects all the persistent elements from all the look and
663        say sequences generated by the given seeds. The string(s) entered as the 
664        parameter(s) will be used as the seed(s) for generating the elements.

686    def get_decay_matrix(self):
687        """
688        Returns the decay matrix as a nested list of integers 
689        (i.e. a list of the rows).
690        The order of the columns and rows correspond to the order
691        in the list of elements.
692        """
693        mat = []
694        e = self.elements
695        for i in range(len(e)):
696            row = []
697            for j in range(len(e)):
698                row.append(e[j].get_decay().count(e[i]))
699            mat.append(row)
700        return mat

702    def get_max_eigenvalue(self):
703        """
704        Returns the maximal real eigenvalue of the decay matrix.
705        In the standard case, this will give Conway's constant. 
706        In general, this will give the growth rate of the look and say sequence. 
707        This method assumes the existence of a real eigenvalue which
708        is larger than (the absolute value) of every other eigenvalue.
709        This assumption is usually guaranteed by the Perron-Frobenius Theorem.
710        """
711        assert len(self.elements) > 0, "The get_max_eigenvalue method requires a nonempty list of elements.\n\tTo fix: Use the generate_elements method prior to calling get_max_eigenvalue."
712        eigenstuff = numpy.linalg.eig(numpy.array(self.get_decay_matrix()))
713        eigenvalues = eigenstuff[0]
714        return max(eigenvalues).real

716    def get_char_poly(self, factor = True, latex = False):
717        """
718        Returns the characteristic polynomial of the decay matrix using sympy.
719        By default the returned polynomial will be factored. 
720        Use ``factor = False`` to get the expanded (i.e. unfactored) polynomial. 
721        Use ``latex = True`` to return the polynomial formatted in latex.
722        """
723        chi = sympy.Matrix(self.get_decay_matrix()).charpoly()
724        if factor:
725            chi = sympy.factor(chi.as_expr())
726        else:
727            chi = chi.as_expr()
728        if latex:
729            return sympy.latex(chi)
730        else:
731            return chi

733    def _get_abundances(self, dec_places = 7, abundance_sum = 100):
734        """
735        Returns a list of relative abundances of each element.
736        By default the abundances are given as percentages, 
737        so they will differ from Conway's abundances by a factor of \\(10^4\\).
738        The abundances can be renormalized by setting the parameter ``abundance_sum``.
739        The order of the list corresponds to the order of the list of elements.
740        """
741        eigenstuff = numpy.linalg.eig(numpy.array(self.get_decay_matrix()))
742        eigenvalues = eigenstuff[0]
743        eigenvectors = eigenstuff[1]
744        index = numpy.where(eigenvalues == max(eigenvalues))
745        limiting_eigenvector_nparray = eigenvectors[:,index].real
746        # The next two lines are converting the numpy array to a list
747        limiting_eigenvector = limiting_eigenvector_nparray.tolist()
748        limiting_eigenvector = [elt[0][0] for elt in limiting_eigenvector]
749        abundance = [abs(round(abundance_sum * num / sum(limiting_eigenvector), dec_places)) for num in limiting_eigenvector]
750        return abundance
